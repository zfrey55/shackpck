🔧 TROUBLESHOOTING:

If npm is not recognized:
- Restart your computer
- Reopen PowerShell
- Try: node --version

If port 3000 is in use:
- Try: npm run dev -- -p 3001
- Then visit: http://localhost:3001

If you see errors:
- Make sure you're in the coins folder
- Try: npm install
- Then: npm run dev
